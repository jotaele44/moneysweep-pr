# Weekly Watch Source Registry

This document describes the additive weekly watch layer for Puerto Rico public-money intelligence.

## Registry files

- `registries/watch_sources.csv`
- `registries/watch_accounts.csv`

## Operating rule

The watch layer is contextual. It does not replace the canonical source registry. Records still require source lineage, validation, and review before entering canonical outputs.

## Social and media handling

Official social accounts and media outlets are lead sources only. They can create project aliases, municipality leads, and follow-up tasks. They cannot establish contract truth by themselves.

## Weekly review buckets

- canonical candidate review
- source document review
- municipal notice review
- social and media lead review
